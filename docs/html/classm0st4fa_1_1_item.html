◆ dotPos

template<typename ProductionT >
pos_t m0st4fa::Item< ProductionT >::dotPos = 0

Dot position rules: 0 => the dot is at the beginning of the body, just before the first symbol. This also means the dot is before the 0th element. n, n > 0 && n < prodSize => the dot is before the nth element. prodSize => the dot is at the end of the production, after the last symbol. The dot position does no take non-grammar-symbol elements in mind.
