\hypertarget{PProduction.h_autotoc_md10}{}\doxysubsubsection{\texorpdfstring{Algorithm\+:}{Algorithm:}}\label{PProduction.h_autotoc_md10}
For every grammar symbol {\ttfamily GS} of the string {\ttfamily S}\+:
\begin{DoxyItemize}
\item If {\ttfamily S} is a non-\/terminal\+:
\item Get FIRST(\+GS) and add it to {\ttfamily FIRST(\+S)}.
\item Check whether FIRST(\+GS) has epsilon and remove the epsilon, if any\+:
\begin{DoxyItemize}
\item If FIRST(\+GS) has epsilon check to see if this is the last symbol of the string\+:
\begin{DoxyItemize}
\item If it does, add epsilon to FIRST(\+GS)
\item If it doesn\textquotesingle{}t, continue to the next symbol.
\end{DoxyItemize}
\item If {\ttfamily FIRST(\+S)} does not have epsilon, we are done with {\ttfamily FIRST(\+S)}.
\end{DoxyItemize}
\item If {\ttfamily S} is a terminal, check to see if it is epsilon\+:
\begin{DoxyItemize}
\item If it is epsilon, check to see if this is the last symbol of the string\+:
\begin{DoxyItemize}
\item If it is, add epsilon to FIRST(\+GS).
\item If it is not, continue to the next symbol.
\end{DoxyItemize}
\item If it is not epsilon\+:
\begin{DoxyItemize}
\item Add the terminal to FIRST(\+GS) and break.
\end{DoxyItemize}
\end{DoxyItemize}
