Algorithm Initialize the stack to contain only the start state. Loop inifinitly until\+: the parser accepts or an error is produced. For every state I on top of the stack and token T from the input\+: A = ACTION\mbox{[}I\mbox{]}\mbox{[}T.\+as\+Terminal\mbox{]} and switch on A\+: If A == SHIFT J\+: Push J on top of the stack and get next input. Else if A == REDUCE J\+: Call \+\_\+reduce(\+J). Else ERROR.

\+\_\+reduce(\+J)\+: Get to production P indexed J and do the following\+: If P contains an action, execute the action before reduction. Get the length L of the production body, and remove from top of the stack L states. Assume the state on top of the stack is now state I, and the head of production J is nonterminal H\+: Push GOTO\mbox{[}I\mbox{]}\mbox{[}H.\+as.\+non\+Terminal\mbox{]} on top of the stack. Check for whether the new top state is an error. \+\_\+error\+\_\+recovery(error\+Recovery\+Type)\+: TBD.
