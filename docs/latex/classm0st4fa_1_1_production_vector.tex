Calculates (if not already calculated) the FOLLOW set of {\ttfamily non\+Terminal} using productions in this production vector. 

This function checks for whether FOLLOW({\ttfamily non\+Terminal}) is already calculated. If it is, it returns it, otherwise, it calculates it and then chaches it to speed up further access, as the process is time-\/ and resource-\/consuming.~\newline
FOLLOW({\ttfamily non\+Terminal}) is calculated using productions of this vector. This is done using the typical algorithm\+:
\begin{DoxyEnumerate}
\item Go through every production and search for {\ttfamily non\+Terminal}.~\newline

\item Add FIRST({\ttfamily S}), assuming {\ttfamily S} is the rest of the production after it.~\newline

\item If {\ttfamily S} is epsilon, then add FOLLOW({\ttfamily H}) assuming {\ttfamily H} is the head of the production being examined. 
\begin{DoxyParams}{Parameters}
{\em non\+Terminal} & The non-\/terminal whose FIRST set is to be calculated. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily true} if FIRST is calculated, otherwise, calculates FIRST and then returns {\ttfamily true}. 
\end{DoxyReturn}

\end{DoxyEnumerate}\hypertarget{PProduction.h_autotoc_md5}{}\doxysubsection{\texorpdfstring{General Algorithm\+:}{General Algorithm:}}\label{PProduction.h_autotoc_md5}

\begin{DoxyItemize}
\item Loop through every production of the grammar until you cannot add another terminal to the FOLLOW set of any non-\/terminal.
\item Variable added asserts whether we added any terminal in the last iteration to the FOLLOW set of any non-\/terminal.
\item For every production {\ttfamily P} and its head {\ttfamily H}\+:
\begin{DoxyItemize}
\item For every grammar symbol {\ttfamily S} of the body of {\ttfamily P}\+:
\begin{DoxyItemize}
\item If {\ttfamily S} is a terminal, get to the next grammar symbol.
\item If {\ttfamily S} is a non-\/terminal call {\ttfamily get\+FOLLOWOf\+Non\+Terminal}.
\end{DoxyItemize}
\end{DoxyItemize}
